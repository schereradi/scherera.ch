# scherera.ch

- **Framework**: [Next.js](https://nextjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com)

## Local development

This application requires Node.js v16.13+.

```bash
git clone https://github.com/schereradi/scherera.ch.git
cd scherera.ch
npm install
npm run dev
```
